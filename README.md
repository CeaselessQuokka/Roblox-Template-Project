# Template Project
A simple template project for Roblox that includes basic tooling such as a Luau language server (Luau LSP) and Rojo. As well as a basic server and client loader.

The project also includes snippets I commonly use. Of course you can delete these as you see fit.

To get started, download the project (or git clone it), and run `./install.ps1` in the terminal.

There is an `extensions.json` file that should hopefully prompt you in VS Code automatically.

Otherwise you will need to download the [StyLua](https://marketplace.visualstudio.com/items?itemName=JohnnyMorganz.stylua) and [Luau LSP](https://marketplace.visualstudio.com/items?itemName=JohnnyMorganz.luau-lsp) extensions.

Note: I would recommend checking `./vscode/settings.json` to ensure everything is to your liking since these are my personal preferences.
